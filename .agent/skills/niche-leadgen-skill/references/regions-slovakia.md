# Zoznam miest pre `--region all-slovakia`

Tento súbor obsahuje mestá, ktoré `discovery.ts` iteruje pri použití `--region all-slovakia`.
Upravuj podľa potreby — každý riadok so `- ` je jeden región.

## Krajské mestá (8)

- Bratislava
- Trnava
- Trenčín
- Nitra
- Žilina
- Banská Bystrica
- Prešov
- Košice
